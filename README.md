# Triplet STDP Spiking Neural Network

Verilog implementation of triplet STDP (Pfister & Gerstner, 2006) for a 2-layer SNN that classifies 5x5 binary images of digits '0' and '1'. Includes four hardware optimization strategies and ASIC synthesis targeting SkyWater 130nm.

## Repository Structure

| Directory | Purpose |
|-----------|---------|
| `rtl/` | Verilog RTL source files |
| `tb/` | Testbenches |
| `lut/` | Pre-computed LUT-STDP ROM tables (generated by Python, used by `snn_dynamic_optimized.v`) |
| `mem-generation/` | Scripts for generating memory initialization files |
| `ppa/` | ASIC synthesis flow (LibreLane/Sky130), regression suite, and PPA analysis notebooks — see [`ppa/README.md`](ppa/README.md) |
| `viz/` | Weight heatmap visualization scripts |
| `results/` | Experiment outputs (weight maps, spike counts, waveforms) |
| `ref/` | Paper, references, and milestone documents |

## RTL Modules

**`rtl/snn_network.v`** — Static pair-based STDP (HW4 baseline). 2-bit weights, shift-register spike history, IF neurons.

**`rtl/triplet_snn.v`** — Static triplet STDP. 4-bit weights, decaying trace variables, LIF neurons, asymmetric A3 parameters.

**`rtl/snn_dynamic.v`** — Parameterized design that can replicate both modules above. All architecture choices exposed as parameters. This is the primary module used for pair vs. triplet experiments.

**`rtl/snn_dynamic_optimized.v`** — Optimized triplet design with four independent optimization strategies controlled by compile-time parameters: LUT-based STDP (`LUT_STDP_EN`), nearest-neighbor trace simplification (`MODE=1`), approximate segmented adder (`SEG_ADDER_EN`), and spike-gated clock enable (`SPIKE_GATE_EN`). Used for PPA analysis and optimization regression.

```bash
python3 -m venv venv

```

## Running the Parameterized Model (`snn_dynamic.v`)

### Train + Test (ablation)

```bash
make ablation                          # default: triplet STDP, 4-bit, LIF
make ablation TRIPLET_EN=0             # pair-based STDP (same architecture, triplet terms off)
```

### With custom spike patterns

```bash
make ablation TRIPLET_EN=1 SPIKE_WHITE=0100000000010000000001000000000100000000 SPIKE_BLACK=0100100001001000010010000100100001001000
```

### Weight heatmap visualization

```bash
make plot-dynamic                      # default triplet config
make plot-dynamic TRIPLET_EN=0         # pair-based
make plot-dynamic SPIKE_WHITE=... SPIKE_BLACK=...   # custom patterns
```

### Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| `W_BITS` | 4 | Weight bit-width (2 = HW4 original, 4 = extended) |
| `TRACE_BITS` | 4 | Trace register width |
| `TRIPLET_EN` | 1 | 1 = triplet STDP, 0 = pair-based only |
| `MODE` | 0 | 0 = all-to-all, 1 = nearest-neighbor |
| `LEAK_EN` | 1 | 1 = LIF, 0 = IF |
| `SYMMETRIC` | 1 | 1 = A2_PLUS = A2_MINUS, 0 = asymmetric pair params |
| `SPIKE_WHITE` | (4 spikes, evenly spaced) | 40-bit spike train for white pixels |
| `SPIKE_BLACK` | (14 spikes, doublet bursts) | 40-bit spike train for black pixels |

#### Optimization parameters (`snn_dynamic_optimized.v` only)

| Parameter | Default | Description |
|-----------|---------|-------------|
| `SPIKE_GATE_EN` | 0 | 1 = gate STDP computation on idle synapses |
| `SEG_ADDER_EN` | 0 | 1 = approximate segmented carry-kill adder for weighted sum |
| `LUT_STDP_EN` | 0 | 1 = ROM lookup replaces STDP multipliers (requires `lut/` files) |

### Named presets

```bash
make ablation-original    # replicates snn_network.v (2-bit, pair, IF)
make ablation-pair        # trace-based pair STDP (4-bit, LIF)
make ablation-triplet     # triplet STDP (4-bit, LIF)
```

## PPA Analysis & Optimization

The `ppa/` directory contains the full ASIC synthesis and optimization evaluation flow. Eight architecture variants (3 baselines + 5 optimized) are synthesized targeting SkyWater 130nm via LibreLane and validated against a 17-pattern functional regression suite. See [`ppa/README.md`](ppa/README.md) for setup and usage.

**Note:** Dynamic power analysis is a work in progress and not yet complete.
